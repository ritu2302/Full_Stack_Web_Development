Pointers for Block

  1. Width of a block level element is 100% of the viewport width
  2. Height of a block isn't set by default
  3. Height & width can be explicitly set using CSS

Pointers for Inline

  1. Width of a inline element is dependent on the content
  2. Height of a inline isn't set by default
  3. Height & width can be explicitly set using CSS
